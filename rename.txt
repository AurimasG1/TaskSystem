⭐ RENAME CHECKLIST (egzaminui paruoštas)
🟩 1. Pervadinti pagrindinį projekto folderį
Pvz.:
MovieCatalog → BookCatalog  
Paaiškinti šį žingsnį

🟩 2. Pervadinti .sln failą
Pvz.:
MovieCatalog.sln → BookCatalog.sln  
Paaiškinti solution rename

🟩 3. Pervadinti projektų folderius
Pvz.:
MovieCatalog.API → BookCatalog.API  
MovieCatalog.Services → BookCatalog.Services  
MovieCatalog.Data → BookCatalog.Data  
Paaiškinti projektų folderių rename

🟩 4. Pervadinti .csproj failus
Atidaryti .csproj ir pakeisti:
<RootNamespace> ir <AssemblyName>  
Paaiškinti csproj rename

🟩 5. Pervadinti ENTITIES
Pvz.:
Movie → Book  
Director → Author  
Genre → Category  
MovieDirector → BookAuthor  
Paaiškinti entity rename

🟩 6. Pervadinti DTO
Pvz.:
MovieRequestDto → BookRequestDto  
MovieResponseDto → BookResponseDto  
MovieUpdateRequestDto → BookUpdateRequestDto  
Paaiškinti DTO rename

🟩 7. Pervadinti SERVICES
Pvz.:
IMovieService → IBookService  
MovieService → BookService  
Metodai:
GetMovieById → GetBookById  
Paaiškinti service rename

🟩 8. Pervadinti REPOSITORIES
Pvz.:
IMovieRepository → IBookRepository  
MovieRepository → BookRepository  
Paaiškinti repository rename

🟩 9. Pervadinti CONTROLLERIUS
Pvz.:
UserMoviesController → UserBooksController  
AdminMoviesController → AdminBooksController  
Paaiškinti controller rename

🟩 10. Pervadinti DbSet’us DbContext’e
Pvz.:
DbSet<Movie> Movies → DbSet<Book> Books  
DbSet<MovieDirector> MovieDirectors → DbSet<BookAuthor> BookAuthors  
Paaiškinti DbSet rename

🟩 11. Pervadinti many‑to‑many ryšį
Pvz.:
MovieDirector → BookAuthor  
MovieId → BookId  
DirectorId → AuthorId  
Paaiškinti many_to_many rename

🟩 12. Pervadinti namespace visame projekte
VS Code:
Ctrl + Shift + F → Replace All  
Ieškai:
MovieCatalog.  
Keiti į:
BookCatalog.  
Paaiškinti namespace rename

🟩 13. Pervadinti migration failus (nebūtina)
Tik jei nori, kad atrodytų gražiai.
Paaiškinti migration rename

🟩 14. Paleisti projektą ir patikrinti
Code
dotnet build
dotnet ef database update
Paaiškinti testavimą po rename
